## 📊 Journal Fields

Each trade captures:
- **Date & Time of entry**
- **Session**: NY AM, NY PM, CBDR, London/Asia, London Open, London Close, NY Launch
- **Pair**: All Forex majors/minors, Crypto, CFDs, Indices
- **Direction**: LONG / SHORT
- **Entry, SL, TP** with auto-calculated R:R and pips
- **Risk Management**: Capital, % risk, lot size
- **Setup / Reason for entry** (text field)
- **Psychology tags**: Confident, FOMO, Revenge Trade, Fear, Disciplined, Off Setup...
- **Rule compliance**: Setup respected? Risk management? SL placed correctly?
- **Screenshot** (Starter & Pro plans)

## 🌐 Custom Domain Setup (journifyx.com)

### Step 1 — Add CNAME to repo
The `CNAME` file in this repo already contains `journifyx.com`.

### Step 2 — Configure DNS at your registrar
Add these records:

| Type | Host | Value |
|------|------|-------|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| CNAME | www | YOUR_USERNAME.github.io |

### Step 3 — Enable HTTPS
In GitHub repo → Settings → Pages → ✅ Enforce HTTPS
